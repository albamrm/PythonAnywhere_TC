Penguin Prediction API

Welcome to the Penguin Prediction API. Use the endpoints to predict penguin species or retrain the model.

Predict

Send a GET request to /api/v1/predict with the following parameters:

island=0&bill_length_mm=50.0&bill_depth_mm=15.0&flipper_length_mm=220&body_mass_g=5000&sex=1

Retrain

Send a GET request to /api/v1/retrain to retrain the model using the available dataset.
